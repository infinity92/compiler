# Winter Programming Language

## How to use 
It is easy. Clone project and run it in xcode and then write text your programm

## Usage
Compiller can it:
 - Create variable: `var a = 10`, `var a = true` (String type not support)
 - Math operations: `+`,`-`,`*`,`/`
 - Priority operators: `(`, `)`
 - Logic operators: `==`, `!=`, `<=`,`>=`,`>`,`<`
 - Scope operator: `{`, `}`
 - Cycles: 
 ```
 var result = 0
 for i = 1 to 10 
 { 
     result = result + i 
 }
 ```
 
 - Conditions: 
 ```
 a = 1
 if (a <= 0) {
    a = a + 1
 } else {
    a = a - 1
 }
 ```
